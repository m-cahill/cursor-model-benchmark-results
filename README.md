# Cursor Model Benchmark — M03 Bounded Results

> This publication presents bounded descriptive observations from one frozen benchmark schedule. It does not establish a general ranking of Cursor models, statistical superiority, or a model recommendation.

## Scope and purpose

This public repository contains the approved bounded descriptive results from the M03 scored screening campaign: four Cursor model selections, four frozen tasks, and three repetitions (48 scored slots). It is an intentionally minimal publication package — not a mirror of the private benchmark implementation.

## Methodology summary

- Headless single-shot executions with fresh workspaces per slot
- First-pass acceptance as the primary metric
- Clopper–Pearson exact 95% confidence intervals
- Hidden evaluator boundary (contents not published)
- Alphabetical presentation by requested model slug

Read the full report: [M03_CURSOR_MODEL_BENCHMARK_BOUNDED_RESULTS.md](M03_CURSOR_MODEL_BENCHMARK_BOUNDED_RESULTS.md)

## Interpretation limits

- Results apply only to the frozen M03 schedule and environment
- Small sample sizes; wide confidence intervals
- T03 recorded uniform nonacceptance; retained in denominators with explicit caveat
- No ranking, superiority claim, or model recommendation is authorized
